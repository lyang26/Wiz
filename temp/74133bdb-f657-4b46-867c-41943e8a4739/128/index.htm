Redis简介？
Redis（nosql一种 )是一个速度非常快的非关系数据库（non-relationaldatabase),它可以储存五种不同类型的值（STRING（字符串）、LIST（列表）、SET（集合）、HASH（散列）和ZSET（有序集合）)，一般常驻在内存当中方便数据获取，同时它可以将内存中的键值持久化到硬盘中。
为什么使用redis？

Redis 特点？
1、开源
2、key-value系统存储服务
3、远程数据结构
4、高性能、功能丰富

Redis特性?
1、速度快
2、持续化
3、多重数据结构
4、支持多种变成语言
5、功能丰富
6、简单
7、主从复制
8、高可用、分布式
Redis应用场景？
缓存系统、排行榜应用、计数器应用、社交网络、消息队列系统、实时系统

Redis 字符串命令：
Set key value   : 指定键设置值  ： SET KEY_NAME VALUE
Get key     键对应的值
getrange key start end  得到字符串的字符串存放在一个键
getset key value 设置键的字符串值，并返回旧值（重新赋值）
getbit key offset 返回储存在键位值得字符串值得偏移
mget key1 key2 [] 获取给定的多个键值

jedis（java连接redis）使用？
maven引用
  1. <dependency>  
  2.     <groupId>redis.clients</groupId>  
  3.     <artifactId>jedis</artifactId>  
  4.     <version>2.0.0</version>  
  5. </dependency>  
spring.xml中配置    <!-- jedis pool配置 -->
    <bean id="jedisPoolConfig" class="redis.clients.jedis.JedisPoolConfig">
        <property name="maxActive" value="${redis.maxActive}" />
        <property name="maxIdle" value="${redis.maxIdle}" />
        <property name="maxWait" value="${redis.maxWait}" />
        <property name="testOnBorrow" value="${redis.testOnBorrow}" />
    </bean>
java 调用:
get（Object key):
    object = redisTemplate.execute(new RedisCallback<Object>() {
            public Object doInRedis(RedisConnection connection) throws DataAccessException {
                byte[] key = keyf.getBytes();
                byte[] value = connection.get(key);
                if (value == null) {
                    return null;
                }
                return toObject(value);
            }
        });  
set(Object key, Object value, final long liveTime) ;
public void set(Object key, Object value, final long liveTime) {

        final String keyf = (String) key;
        final Object valuef = value;
        // final long liveTime = 86400;
        redisTemplate.execute(new RedisCallback<Long>() {
            public Long doInRedis(RedisConnection connection) throws DataAccessException {
                byte[] keyb = keyf.getBytes();
                byte[] valueb = toByteArray(valuef);
                connection.set(keyb, valueb);
                if (liveTime > 0) {
                    connection.expire(keyb, liveTime);
                }
                return 1L;
            }
        });
